CSS, que significa Cascading Style Sheets (Hojas de Estilo en Cascada), es un lenguaje de diseño gráfico utilizado para estilizar elementos escritos en lenguajes de marcado como HTML o XML. Su función principal es separar el contenido de la presentación visual de una página web, permitiendo a los desarrolladores y diseñadores controlar aspectos como colores, fuentes, disposición de elementos, fondos y animaciones sin alterar la estructura del documento. Esta separación mejora la accesibilidad, facilita el mantenimiento y actualización del diseño, y permite crear sitios web más atractivos y adaptables a diferentes tamaños de pantalla. CSS fue desarrollado por el World Wide Web Consortium (W3C) en 1996 para resolver los problemas de formato que surgían al usar HTML solo para estructurar contenido, ya que este no estaba diseñado para incluir etiquetas de estilo.
