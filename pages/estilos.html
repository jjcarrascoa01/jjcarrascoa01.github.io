HTML, que significa Lenguaje de Marcado de Hipertexto (HyperText Markup Language), es el lenguaje estándar utilizado para crear y estructurar páginas web. Sirve para definir el contenido y la estructura de una página web mediante el uso de etiquetas que indican a los navegadores cómo interpretar y presentar elementos como texto, imágenes, enlaces, vídeos y otros recursos multimedia.

CSS, o Cascading Style Sheets (Hojas de Estilo en Cascada), es un lenguaje de diseño gráfico que se utiliza para estilizar y controlar la presentación visual de documentos escritos en lenguajes de marcado como HTML o XML. Su principal función es separar el contenido del diseño, permitiendo a los desarrolladores y diseñadores definir aspectos como colores, fuentes, disposición de elementos, animaciones y efectos visuales sin alterar la estructura semántica del contenido.
